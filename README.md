## ✨ Características

- 🎬 **24 películas** con póster, título, puntuación y sinopsis
- 🔍 **Búsqueda dinámica** en tiempo real con debounce
- 📄 **Paginación** - 12 películas por página
- 🎨 **Diseño oscuro moderno** con animaciones suaves
- 📱 **100% Responsive** (móvil, tablet, escritorio)
- 🔄 **Doble modo**: Live (API) / Simulado (datos locales)
- 🔒 **API Key protegida** (no visible en producción)
- 💀 **Skeleton loading** mientras carga
- ⚡ **Rendimiento optimizado** (Vanilla JS, sin frameworks)
